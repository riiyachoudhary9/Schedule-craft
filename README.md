# 📅 ScheduleCraft

ScheduleCraft is a modern web-based timetable generator designed to help students and individuals create, manage, and export personalized schedules with ease.

Built entirely using **HTML, CSS, and JavaScript**, the application provides an intuitive interface for generating academic and personal timetables, saving history, and exporting schedules in multiple formats.
